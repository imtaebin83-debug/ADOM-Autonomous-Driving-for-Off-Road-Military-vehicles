# ROS2 Packages

ROS2 package skeletons will be added here when the first node boundary is decided.

Candidate packages:

- `adom_perception_ros`: camera image to segmentation mask
- `adom_costmap_ros`: segmentation/depth to semantic costmap
- `adom_bringup`: launch files and runtime configs
- `adom_msgs`: custom messages, if standard messages are not enough
